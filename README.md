# GodoTroll

一个基于 Godot 4 开发的 2D 平台跳跃整蛊游戏。本项目旨在通过经典的平台跳跃玩法，展示复杂的“阻拦系统”逻辑以及规范的项目架构。

## 🎮 プロジェクトの概要 / 项目概要
* **Engine**: Godot Engine 4.x
* **Language**: GDScript (中文 & 日本語のダブルコメント)
* **Genre**: 2D Platformer / Troll Game

## ✨ 特徴 / 项目特点
* **Refactored Architecture**: 完全に整理されたディレクトリ構造。
* **Hindrance System**: 多様な障害物（ボム、落とし穴、トロール要素）の実装。
* **Modular Design**: 敵キャラやギミックを独立したモジュールとして設計。

## 📂 ディレクトリ構造 / 目录结构
* `res://assets/`: 音声、画像などのリソース。
* `res://entities/`: プレイヤー、敵、障害物などの実体。
* `res://levels/`: ステージのデザインとスクリプト。
* `res://systems/`: グローバルなマネージャー。

## 🚀 実行方法 / 如何运行
1. Godot Engine 4.x をダウンロードします。
2. このリポジトリをクローンまたはダウンロードします。
3. `project.godot` を Godot で開きます。
